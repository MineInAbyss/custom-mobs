MobDrop

data class MobDrop(item: SerializableItemStack, cooked: SerializableItemStack?, cookExp: Float, cookTime: Int, minAmount: Int, maxAmount: Int, dropChance: Double)

A serializable class for defining an item a mob can drop. It acts as a sort of factory for ItemStacks that may define rarity, how many items to spawn, etc...

Also allows defining a cooked version of the item that will automatically be registered when the class is created.

Parameters

item

The item to be spawned.

cooked

The cooked version of this item.

cookExp

The amount of exp to drop when this item is cooked.

cookTime

How long it takes for this item to cook

minAmount

The minimum stack size of this item.

maxAmount

The maximum stack size of this item.

dropChance

A chance from 0 to 1 for this item to be dropped.
